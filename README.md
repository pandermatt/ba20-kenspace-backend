## Developing

### Setup

```bash
git clone git@github.com:pandermatt/ba20-kenspace-backend.git
cd ba20-kenspace-backend
pip install -r requirements.txt
```

### Run
```bash
python app.py
```

#### Configuration

```bash
cp application.example.yml application.yml
```

Fill in all your keys
